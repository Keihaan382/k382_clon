## Description
The `client.lua` script is designed for a gaming environment where players can generate clones of their character with various animations and props. This script allows players to interact with the environment in a more dynamic and fun way, using predefined animations that can be invoked through in-game commands.

## Functionality
The script allows players to:

- **Generate clones** of their character at their current location.
- **Execute specific animations** on the clones, which may include actions such as sitting, playing the guitar, dancing, among others.
- **Use props** (objects) that can be attached to the clones during the execution of the animations, such as umbrellas, guitars, and more.

## Commands
The script includes the following commands that players can use:

1. **Generate a clone with one animation**:
   - Command: `/k [animation_name]`
   - Example: `/k sit` will generate a clone that sits.

2. **Generate a clone with two animations**:
   - Command: `/k [animation_name_1] [animation_name_2]`
   - Example: `/k sit guitar` will generate a clone that sits while playing the guitar.

3. **Deactivate spawn mode** (optional):
   - Command: `/k2`
   - This command deactivates spawn mode, although it is not necessary for the main functionality of the script.

## Available Animations
The script includes a wide variety of animations that players can use. Some of the available animations are:

- **Actions with props**:
  - `["umbrella"]` - Umbrella
  - `["guitar"]` - Guitar
  - `["coffee"]` - Coffee
  - `["book"]` - Book

- **Movement actions**:
  - `["jog2"]` - Jog 2
  - `["jumpingjacks"]` - Jumping Jacks
  - `["pushup"]` - Push-ups

- **Interaction actions**:
  - `["knock"]` - Knock
  - `["salute"]` - Salute
  - `["dance"]` - Dance

Players can combine these animations to create unique and fun interactions in the game.

## Usage
To use the script, players must ensure that the script is correctly loaded into their gaming environment. Once the script is active, they can use the commands mentioned above to generate clones and execute animations.

### Example of Use
1. To generate a clone that sits, the player can type:
   ```
   /k sit
   ```
2. To generate a clone that sits and plays the guitar, the player can type:
   ```
   /k sit guitar
   ```
